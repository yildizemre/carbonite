# Carbonited

Kurumsal karbon kredisi pazaryeri — fiziksel kredilerin tokenizasyonu ve anlık likidite platformu.

## Kurulum

```bash
npm install
npm run dev
```

Uygulama `http://localhost:5173` adresinde çalışır.

## Build

```bash
npm run build
npm preview
```

## Sayfalar

| Sayfa | Route |
|-------|-------|
| Pazaryeri | `#/` |
| Proje Detayı | `#/proje/:id` |
| Satın Al | `#/proje/:id/satin-al` |
| Portföy | `#/portfoy` |

Giriş yap: [app.carbonited.com](https://app.carbonited.com)
